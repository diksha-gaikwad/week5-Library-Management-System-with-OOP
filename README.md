📚 Library Management System (OOP Based)

A Python-based Library Management System built using Object-Oriented Programming (OOP) principles.
This project demonstrates real-world system design using classes, objects, file handling, and modular architecture.

🚀 Features

📖 Add and manage books

👤 Register library members

🔄 Borrow and return books

⏳ Automatic due date calculation (14-day loan period)

💾 JSON-based data persistence

📦 Modular package structure

🧪 Basic unit testing

🖥️ Command-line menu interface


🧠 OOP Concepts Used

Classes & Objects

Encapsulation

Constructors (__init__)

Class vs Instance Variables

Modular Programming

File Handling (JSON)

Basic Testing

⚙️ How to Run the Project

1️⃣ Navigate to Project Folder
cd week5-library-management-system
2️⃣ (Optional) Create Virtual Environment
python -m venv venv
venv\Scripts\activate
3️⃣ Run the Application

⚠️ Important: Run from the root folder (outside library_app)

python -m library_app.main
🧪 Run Tests
pip install pytest
pytest
